# TreasureMazeRun
Game i made entirely using c and python integration for a solo Hackathon project. Originally uploaded to gitlab. I recommend checking it out. This is one that I actually feel proud in because it was a challenging project with little help. Updated With UI features and signals for feedback. The user creates their own profile and that saves their stats each game. Collect all Treasure throughout the maze and go through portals to collect from other rooms until all treasure is found. Please run after making sure you are in the correct folder ( if not cd into the folder, this can be confirmed using ls to see assets c python etc), then use make dist to connect libraries and python3 run_game.py --config ../assets/starter.ini --profile ../assets/myprofile.json to create your profile and run the game, enjoy!
